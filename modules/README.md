# Fouroneone status codes
* `0`: `>dial *411` entry
* `1`: 411 option 1 (Check a number)
  * `10`: Result
* `2`: 411 option 2 (Search by query)
  * `20`: Result
* `3`: 411 option 3 (Change registry)
  * `30`: Registry changed
  * `31`: Registry deleted
* `4`: `>dial *233`
* `5`: `>wizard` for guilds
* `6`: `>wizard` for DM numbers
